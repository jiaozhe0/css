## 变量定义
### 颜色
+ 主色: @color-primary
+ 副色: @color-deputy
+ 成功: @color-success
+ 危险：@color-danger
+ 信息：@color-info
+ 警告：@color-wraning
+ 提示：@color-hint
+ 背景色：@color-bg
+ 分割：@color-split
+ 激活色: @color-[]-active
+ 一级字体： @color-text
+ 二级字体： @color-text-secondary
+ 三级字体： @color-text-gray

### 字体大小
~~~ 
 单位: 文本元素字体大小使用em,  模块使用rem
 100% = 16px = 1rem
 62.5% = 10px = 1rem
~~~
+ 正常字体： @font-size 16px = 1.6rem = 62.5%
+ 次小级： @font-size-sm
+ 次次小级： @font-size-xs
+ 次次次小级：@font-size-xxs
+ 次大级：@font-size-lg
+ 次次大级：@font-size-xl
+ 次次次大级：@font-size-xxl

### 间距
+ 正常：@margin
+ 次小：@margin-sm
+ 次次小 @margin-xs
+ 次大：@margin-lg
+ 次次大：@margin-xl
